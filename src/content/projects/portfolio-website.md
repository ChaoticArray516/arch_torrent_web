# Portfolio Website

This is a sample project entry for the portfolio website.

## Overview

Built with Astro's content-first philosophy, this portfolio demonstrates:

- **Zero JavaScript by default**: Static HTML/CSS for maximum performance
- **Islands Architecture**: Interactive components only where needed
- **Type-safe content**: Zod-validated frontmatter via Content Collections
- **Build-time optimization**: Images processed at build, not runtime

## Features

- Responsive design with Tailwind CSS
- Dark/light theme toggle
- Project filtering with React islands
- MDX content support for rich documentation
